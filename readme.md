# Fairway Forge Media

**Fairway Forge Media** is a St. Louis-based technical media outlet dedicated to reverse-engineering the intersection of professional sports and emerging technology. 

We move beyond the leaderboard to document the hardware, infrastructure, and "Maker" innovation that powers the modern PGA Tour experience.

## Tech Stack
- **Framework:** [Hugo/Next.js/etc]
- **Deployment:** [Netlify/Vercel]
- **Design:** Tailwind CSS

## Mission
To reverse-engineer the intersection of professional sports and emerging technology, documenting the hardware and "Maker" elements of the PGA Tour.
